# lwc

There's nothing to read about.

## Usage
```
A wc clone

Usage: lwc [OPTIONS] <ENTRIES>...

Arguments:
  <ENTRIES>...
          Target files

Options:
  -r, --recursive
          Traverse directories recursively
  -o, --oneline
          Display stats in one line
  -f
          Count elements in a directory
  -a, --absolute-paths
          Print paths as absolute paths
  -h, --help
          Print help
  -V, --version
          Print version
```
